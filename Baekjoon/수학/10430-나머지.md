# 나머지
## 문제
(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?

(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?

세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 A, B, C가 순서대로 주어진다. (2 ≤ A, B, C ≤ 10000)

## 예제 입력 1
    5 8 4
  
## 예제 출력 1
    1
    1
    0
    0
    
## 알고리즘 분류
- 수학
- 구현
- 사칙연산

## 풀이
주어진 식 그대로 출력하면 됨.

주의: 곱하기는 x가 아니라 *

## python 코드
```python
A, B, C = map(int, input().split())

print((A+B)%C)
print(((A%C) + (B%C))%C)
print((A*B)%C)
print(((A%C) * (B%C))%C)
```
